# 🌟 Plateforme Web - Club Enactus FSBM

Ce projet consiste à développer la plateforme web officielle du club **Enactus FSBM** de la Faculté des Sciences Ben M'Sik[cite: 2, 9]. [cite_start]Conçue pour centraliser les informations du club, elle permet aux étudiants de découvrir nos projets d'entrepreneuriat social, de suivre nos événements et de rejoindre notre communauté.

Projet réalisé dans le cadre du module **Programmation Web 2 : Langage JavaScript** (Semestre 4 - 2025/2026) [cite: 4, 5] sous la supervision du **Prof. M. Ghazouani**.

---

## 🚀 Fonctionnalités
* **Page d'accueil complète :** Présentation d'Enactus FSBM, logo, vision et objectifs du club.
* **Membres du bureau :** Liste des membres principaux (Président, Vice-Président, Responsables de pôles).
* **Gestion des Événements :** Affichage clair et dynamique des activités et projets prévus.
* **Interaction :** Formulaire / Bouton "Rejoindre le club" entièrement fonctionnel côté client
* **Design Responsive :** Interface moderne, ergonomique et adaptée aux mobiles/tablettes grâce à Tailwind CSS (ou Bootstrap).

---

## 🛠️ Technologies Utilisées
* **Langage :** JavaScript (ES6+) 
* **Framework Frontend :** ReactJS
* **Routage :** React Router 
* **Gestion d'état :** React Hooks (`useState`, `useEffect`) 
* **Style & Animations :** Tailwind CSS / Bootstrap
